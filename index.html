Big O Notation Practice

Step One: Simplifying Expressions

  1. O(n + 10): O(n)
  2. O(100 * n): O(n)
  3. O(25): O(1)
  4. O(n^2 + n^3): O(n3)
  5. O(n + n + n + n): O(n)
  6. O(1000 * log(n) + n): O(n)
  7. O(1000 * n * log(n) + n): O(nlog(n))
  8. O(2^n + n^2): O(2n)
  9. O(5 + 3 + 1): O(1)
  10. O(n + n^(1/2) + n^2 + n * log(n)^10): O(n2)

Step Two: Calculating Time Complexity

  1. O(n)
  2. If n > 10 O(n), if n < 10 O(1)
  3. For n <= 10 O(n) For n > 10 O(1)
  4. O(n)
  5. O(n2)
